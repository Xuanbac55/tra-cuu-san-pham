⚠ Lưu ý quan trọng: sản phẩm bạn thêm ở đây chỉ lưu trên máy này. Trang khách hàng sẽ KHÔNG thấy sản phẩm mới cho đến khi bạn làm xong bước 2 (tải products.json) rồi tải file đó lên GitHub thay cho file cũ.
